# ♟️ Chess Engine - zhayyn™ Edition

A visually stunning, lightweight, browser-based Chess Engine built with HTML, CSS, and Vanilla JavaScript.

## ✨ Features
* **Elegant Glassmorphism UI**: Beautiful deep blue and silver frosted glass design.
* **Smart AI Logic**: The AI prioritizes center-board control and aggressive captures.
* **Move Highlighting**: Distinctive visual feedback for possible moves and AI's last turn.
* **Modular Codebase**: Cleanly separated into `HTML`, `CSS`, and `JS` files for easy learning and scalability.

## 🚀 How to Play
1. Clone this repository to your local machine.
2. Open `index.html` in any modern web browser.
3. You play as the **White Pieces** (bottom). Click on a piece to see legal moves (highlighted in red) and click the destination square to move.
4. Outsmart the AI!

## 🛠️ Built With
* HTML5
* CSS3 (Flexbox, Backdrop-filter for Glassmorphism)
* Vanilla JavaScript (DOM manipulation, Array-based logic)
